Para ejecutar el sistema basta con abrir el ejecutable MoliGT

Requiere una máquina con Windows de 64 bits

Para explorar el proyecto y compilar para otras plataformas se debe utilizar 
la aplicación Godot 3.0 incluida. 

 - Copiar Godot_v3.0-stable_win64.exe a otra localidad
 - Ejecutarlo y dar click en importar
 - Buscar el archivo project.godot

Si se desea mover el programa compilado se debe colocar el archivo MoliGT.pck 
en la misma localidad.
